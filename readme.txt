Title: Intrusion Detection Systemm Setup in Virtual Environment

The Suricata script file will download, install and configure suricata tool in any linux system. With that we can easily detect network attacks. The logs will be automatically stored in the /var/log/suricata directory.

The Webrules script file will add some web rules in order to detect malicious webpages which is being visited.

The Usb scan script file will download a tool and start a process in background so whenever any malicious usb inserted in the linux system, it will scan it automatically and if it includes any malware or virus, it will also generate logs.
